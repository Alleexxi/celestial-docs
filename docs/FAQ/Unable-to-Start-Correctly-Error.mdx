If you encounter an "Unable to Start Correctly" error while using Celestial, follow these steps to troubleshoot the issue:

### Step 1: Reinstall Roblox

Try reinstalling Roblox to see if that resolves the issue.

---

### Step 1.2: Reinstall Celestial

Fully reinstall Celestial by deleting the appdata folder in "%appdata%". Then, download and install Celestial again.

---

### Step 2: Reinstall the Redistributes

Reinstall the redistributes. You can find them in the "appwiz.cpl" program.

---

We hope this guide has helped you resolve the "Unable to Start Correctly" error in Celestial. If you have any further questions or concerns, please do not hesitate to reach out to us.
