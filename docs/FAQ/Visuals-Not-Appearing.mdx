If the visuals are not appearing in Celestial, follow these steps to troubleshoot the issue:

### Step 1: Check Redistributes

Make sure the redistributes are installed. You can find them in the "appwiz.cpl" program.

---

### Step 2: Install DirectX

Install DirectX from the [Microsoft website](https://www.microsoft.com/en-us/download/details.aspx?id=35). This may fix the issue with the visuals not appearing.

---
