If Celestial is stuck in a loop waiting for DLL response, try the following solution:

### Step 1: Enable Discord Game Overlay

Enabling the Discord Game Overlay has been known to resolve the issue of Celestial getting stuck in a loop waiting for DLL response. Follow these steps to enable it:

1. Open Discord and click the User Settings icon in the bottom-left corner of the screen.
2. Click on the Overlay tab in the left-hand menu.
3. Toggle on the "Enable in-game overlay" switch.
4. Restart Discord and launch Roblox.
