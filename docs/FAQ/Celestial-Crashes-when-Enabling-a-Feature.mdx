If Celestial crashes when you try to enable a feature, follow these steps to troubleshoot the issue:


### Step 1: Disable Anti-Virus and Delete Celestial Files

First, completely disable your anti-virus software. Then, delete all Celestial-related files, including the folder in `%appdata%` (hold Win + R, type `%appdata%`, and click OK).

---

### Step 1.2: Redownload and Install Celestial

Once you have deleted all Celestial-related files, redownload the injector and install Celestial again.
