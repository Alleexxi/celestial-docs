If Roblox is not found on the console while using Celestial, follow these steps to troubleshoot the issue:

### Step 1: Run Celestial as Administrator

Try running Celestial as an administrator to see if that resolves the issue. This may give Celestial the necessary permissions to locate Roblox on the console.
