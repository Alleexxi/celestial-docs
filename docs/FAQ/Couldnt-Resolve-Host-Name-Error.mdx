If you encounter a **Couldn't resolve host name** error while using Celestial, follow these steps to troubleshoot the issue:

### Step 1: Use the Latest Injector

Make sure you are using the latest version of the injector. You can download the latest injector from the Celestial website.

---

### Step 2: Run Celestial as Administrator

Try running Celestial as an administrator. Right-click on the Celestial executable and select **Run as administrator**.
