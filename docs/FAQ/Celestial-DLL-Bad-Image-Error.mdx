If Roblox shows an error named "Bad Image" with the name of the Celestial DLL, follow these steps to troubleshoot the issue:

![](/img/faq/badimage.png)

### Step 1: Disable Anti-Virus and Extract Celestial

First, disable your anti-virus software and extract Celestial before running it.

---

### Step 2: Delete Any Anti-Virus Software

Use the `appwiz.cpl` program (Hold Windows + R > type `appwiz.cpl` > Click OK) to look for any anti-virus software installed on your system. If any are found, delete them.

---

### Step 3: Delete the Celestial DLL

Hold Windows + R > type `%appdata%/Celestial` > Click OK and delete the DLL if it's there.

---
