If the Celestial RAR file appears as a Chrome file after downloading, follow these steps to resolve the issue:

### Step 1: Install WinRAR or 7-Zip

Make sure you have either WinRAR or 7-Zip installed on your system. These programs are necessary to extract the contents of the RAR file.

---
### Step 2: Open the RAR File with WinRAR/7-Zip

Right-click the Chrome file and select "Open with WinRAR" or "Open with 7-Zip". This will open the RAR file in the corresponding program, allowing you to extract its contents.
