If Celestial is not appearing after injection, please follow these steps to troubleshoot the issue.

### Step 1: Open Celestial UI

Press the `Insert` or `F1` key to open the Celestial UI. If the UI does not appear, proceed to the next step.

---
### Step 2: Uninstall/Disable Anti-Virus Software

Some anti-virus software may interfere with Celestial's injection process. To check if this is the case, uninstall or disable any anti-virus software installed on your system. You can do this through the `appwiz.cpl` program. If the issue persists, proceed to the next step.

---
### Step 3: Check if Redistributables are Installed

Make sure the redistributables required for Celestial to run are installed on your system. You can check this through the `appwiz.cpl` program. If they are not installed, please install them and try injecting Celestial again. If the issue persists, proceed to the next step.

---
### Step 4: Enable Discord Game Overlay on Roblox

If you have Discord installed, make sure the Discord Game Overlay is enabled for Roblox. This can be done through the Discord settings. If the issue persists, proceed to the next step.

---
### Step 5: Disable Overlays

If you have any overlays running on your system, such as Overwolf or GeForce Experience, disable them and try injecting Celestial again. If the issue persists, please contact Celestial support for further assistance.

---

We hope this guide has helped you resolve the issue with Celestial injection. If you have any further questions or concerns, please do not hesitate to reach out to us.
